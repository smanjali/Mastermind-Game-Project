# Mastermind-Game

## How to run
This game must be run with an emulator. It is **not** posted and can not be downloaded on the app store. I used Android Studio to create and run the app and can provide guidance on running the app only with Android Studio. With this IDE you have two options for playing the game: On your Android phone through wifi or on the provided emulator in Android Studio IDE.

## With an Android
Your android phone should be able to connect to the Android Studio IDE where you can run the app directly on your phone.

## With Android Studio Emulator
### Creating an Emulator on Android Studio
Click the *Device Manager* icon on the right-hand-side of the Android Studio screen (it should look like a phone with a little android robot) or in the toolbar navigate to View > Tool Windows > Device Manager

Click the '+' icon to create a new virtual device or select a device if provided. I recommend using a medium phone with API 35. 

Run the device by clicking the run icon. It will likely take a while to start up as the IDE has to download the entire phone set up. 

Once the phone is set up you can power it on. You can either run the app by clicking the mastermind app icon on the emulator or running the app through Android Studio.

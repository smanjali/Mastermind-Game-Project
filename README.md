# Mastermind-Game

## How to run
This game must be run with an emulator. It is **not** a posted and can not be downloaded on the app store. I used Android Studio to create and run the app. With this IDE you have two options for playing the game depending on if you have an android phone.

## With an Android
Your android phone should be able to connect to the Android Studio IDE where you can run the app directly on your phone.

## With Android Studio Emulator
Open and run an emulator on Android studio. run project.
